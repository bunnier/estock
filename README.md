# 码盯 (estock)

码盯是一个在 VS Code / Cursor 状态栏查看 A 股、港股股票行情的插件。

> 边写代码，边看一眼行情。

## 功能

- 在状态栏同时展示多只股票。
- 支持沪深 A 股和港股。
- 支持股票池管理和状态栏展示列表。
- 点击状态栏股票即可切换展示内容。
- 交易时段自动刷新，休市后显示「休市」。
- 支持手动刷新、查看详情、调整颜色和展示格式。

## 安装

### 从插件市场安装

在扩展面板搜索「码盯」或「estock」，点击安装。

如果 Cursor 扩展面板暂时搜索不到，可以使用命令行安装：

```bash
cursor --install-extension estock.estock-watcher
```

### 从 VSIX 安装

1. 下载 [最新版本的 `.vsix` 文件](https://github.com/bunnier/estock/releases)。
2. 打开命令面板，选择 `Install from VSIX...`。
3. 选择下载好的 `.vsix` 文件，安装后重新加载窗口。

## 使用

安装后，状态栏会显示默认关注的股票。

### 添加股票

打开命令面板，执行 `码盯: 添加股票`，输入股票代码即可添加。

支持的代码格式：

| 市场 | 格式 | 示例 |
|------|------|------|
| A 股 | 6 位数字 | `601318`、`000001` |
| 港股 | 5 位数字 | `00700`、`00388` |

插件会自动识别市场，无需手动输入 `sh`、`sz` 或 `hk` 前缀。

### 切换状态栏股票

有两种方式：

1. 直接点击状态栏上的某只股票，然后从股票池中选择要展示的股票。
2. 打开命令面板，执行 `码盯: 切换状态栏股票`，先选择状态栏位置，再选择股票。

### 查看详情

打开命令面板，执行 `码盯: 查看股票详情`，可以查看当前价、涨跌、今开、最高、最低、成交量和更新时间。

### 手动刷新

打开命令面板，执行 `码盯: 刷新股票数据`，会立即刷新股票行情。

### 移除股票

打开命令面板，执行 `码盯: 移除股票`，选择要移除的股票即可。

## 配置

在设置中搜索 `estock` 可以调整插件行为。

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `estock.watchList` | `["601318", "00700", "002027", "601899"]` | 股票池，保存所有关注的股票。 |
| `estock.displayList` | `["00700", "601318", "002027"]` | 状态栏展示列表，决定显示哪些股票和显示顺序。 |
| `estock.maxDisplay` | `3` | 展示列表为空时，最多从股票池中展示几只股票。 |
| `estock.refreshInterval` | `10` | 自动刷新间隔，单位为秒，最小值为 5。 |
| `estock.dataSource` | `"smart"` | 行情来源，建议保持默认值。 |
| `estock.displayFormat` | `"${name} ${changePercent} (${price})"` | 状态栏展示格式。 |
| `estock.statusBarAlignment` | `"center"` | 状态栏位置，可选 `left`、`center`、`right`。 |
| `estock.colorUp` | `"#FAAFA0"` | 上涨颜色。 |
| `estock.colorDown` | `"#4EC9B0"` | 下跌颜色。 |
| `estock.colorFlat` | `"#CCCCCC"` | 平盘颜色。 |
| `estock.colorClosed` | `"#AAAAAA"` | 休市颜色，收盘后统一使用。 |
| `estock.showVolume` | `true` | 详情中是否显示成交量。 |

### 展示格式

`estock.displayFormat` 支持以下变量：

| 变量 | 说明 | 示例 |
|------|------|------|
| `${name}` | 股票名称，包含市场标识。 | `腾讯控股(H)` |
| `${price}` | 当前价。 | `421.00` |
| `${change}` | 涨跌额。 | `+1.20` |
| `${changePercent}` | 涨跌幅。 | `+0.29%` |
| `${volume}` | 成交量。 | `7849461` |

示例：

```json
"${name} ${changePercent} (${price})"
```

## 交易时间

| 市场 | 自动刷新时段 |
|------|--------------|
| A 股 | 周一至周五 9:15-11:30、13:00-15:00 |
| 港股 | 周一至周五 9:00-12:00、13:00-16:10 |

插件会按 A 股和港股各自的市场时间判断是否刷新。A 股和港股不会互相影响：例如港股休市但 A 股开市时，A 股仍会刷新，港股会显示「休市」。

休市判断会参考内地和香港公众假期。遇到交易所临时休市、特殊交易安排或节假日调整时，实际状态可能以交易所公告为准。

## 数据说明

本插件使用免费的公开行情来源，无需配置账号或密钥。免费行情通常会有延迟，尤其是港股行情，状态栏展示仅用于快速查看，不适合作为交易依据。

默认数据源会自动为 A 股和港股选择较合适的免费行情来源。一般建议保持 `estock.dataSource` 为 `smart`。
